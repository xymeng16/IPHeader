# IPHeader
A simple program to generate normal IPHeader.
## INPUT
The source and destination IP addresses.
## OUTPUT
The content of the IP header, displayed in hex,byte by byte.
Like:  
  
        45 00 00 14  
        D6 67 00 00  
        FF 01 74 11  
        AC 10 0B C8  
        AC 10 0B 87
